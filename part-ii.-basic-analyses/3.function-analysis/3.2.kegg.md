## 3\) Running Steps

1. `Input` -&gt; `Type` 勾选 "Ensembl Gene ID"
2. 将上文中的基因ID粘贴到文本框中
3. `Database` 勾选 "KEGG Pathway"
4. 单击 `Run`

## 4\) Tips/Utilities

KOBAS 也可以进行 GO 富集，只需要在 `Database` 勾选 "GO" 即可

除了 KOBAS 之外，其它软件也能进行 KEGG 分析，如 [DAVID](https://david.ncifcrf.gov/)
